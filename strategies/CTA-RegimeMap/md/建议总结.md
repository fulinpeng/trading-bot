# 一、市场状态识别（Regime Map）

目标：

```
避免在震荡市场交易
```

使用指标：

* Average Directional Index（E:\githubProjects\trading-bot\utils\adx.js）
* Average True Range（E:\githubProjects\trading-bot\utils\atr.js）
* Exponential Moving Average（E:\githubProjects\trading-bot\utils\ma.js的calculateEMA）
* RSI（E:\githubProjects\trading-bot\utils\rsi.js）

---

## 状态分类

### 趋势上涨

条件：

```
ADX(14) > 25
close > EMA200
RSI > 55
```

变量：

```
regime = TREND_UP
```

允许：

```
做空、做多
```

---

### 趋势下跌

条件：

```
ADX > 25
close < EMA200
RSI < 45
```

变量：

```
regime = TREND_DOWN
```

允许：

```
只做空
```

---

### 震荡市场

条件：

```
ADX < 20
ATR < SMA(ATR,50)
```

变量：

```
regime = RANGE
```

处理：

```
不交易
```

---

### 爆发行情

条件：

```
ATR > ATR_MA * 1.5
AND
ADX 上升
```

变量：

```
regime = EXPLOSION
```

允许：

```
breakout 策略
（具体有哪些呢????）
```

# 三、趋势密度模型（Trend Density）

目的：

```
确认趋势持续性
```

计算：

```
trendDensity =
count(close > EMA200, 50) / 50
```

做多条件：

```
trendDensity > 0.6
```

做空条件：

```
trendDensity < 0.4
```

---

# 四、趋势爆发检测（Trend Explosion Detector）

目的：

```
捕捉大行情启动
```

使用：

* Bollinger Bands(E:\githubProjects\trading-bot\utils\boll.js)
* Keltner Channel(E:\githubProjects\trading-bot\utils\BBKeltner.js)

---

### 压缩阶段

```
BB_upper < KC_upper
AND
BB_lower > KC_lower
```

变量：

```
squeezeOn
```

---

### 释放阶段

```
BB_upper > KC_upper
OR
BB_lower < KC_lower
```

变量：

```
squeezeRelease
```

---

### 爆发确认

```
ATR > ATR_MA * 1.3
AND
volume > SMA(volume,20)*1.8
```

最终：

```
trendExplosion
```

# 八、ATR 自适应止损

使用：

* Average True Range

---

### 波动率检测

```
atrRatio = ATR(14) / SMA(ATR,50)
```

---

### 止损倍数

```
atrRatio < 0.8 → trailStop = 2.5 ATR
0.8 < atrRatio < 1.2 → trailStop = 3 ATR
atrRatio > 1.2 → trailStop = 3.5 ATR
```

